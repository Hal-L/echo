String
	String 是属于java.lang包下  ,这个包下面的所有类不需要导包
	String -> 字符串   java中所有被""引起来的都是字符串
	字符串是不可变的
	构造方法
		public String(){}  -- >创建一个空的字符串
		public String(char[] chars) --> 根据一个字符数组创建一个字符串
		public String(String s)   ---> 根据字符串创建一个字符串
		String s = "";    
	== 作用
	在比较基本数据的时候比较的是具体值
	在比较引用数据类型的时候比较的是地址值
	"" 和new 创建字符串的区别
		""创建是存储在字符串常量 ,并且如果常量中已经存在了内容相同的字符串就不会创建新的,直接完成赋值
		new创建字符串是存储在堆内存中.而且是每new一次就开辟一块新的地址值
	字符串的具体内容的比较
		boolean equeal(String s);      -->区分大小写比较字符串内容
		boolean equalsIgnoreCase(String s);  --> 不区分大小写比较字符串内容
		char charAt(int index)    -->根据索引返回字符串中对应的字符
		int length()    --> 返回字符串的长度
		char[] toCharArray()    --> 将字符串每个字符封装到一个字符数组里面
		String substring(int index)    --> 返回指定索引到字符串结束位置的字符串 
		String substring(int begin , int end)  -->截取两索引之间的字符串返回（包含前面的索引 不包含后面的索引）
		// target 被替换的旧字符串 replacement 替换的新字符串
		String replace(CharSequence target, CharSequence replacement)   --> 讲字符串中的target内容替换成replacement 将替换后的结果返回
		String[] split(String regex)  --> 将字符串按照regex字符串进行切割成多个部分.然后把切割出来的每一部分封住到一个字符串数组中返回
StringBuilder 
	作用:提高字符串的操作效率
	 System.currentTimeMillis(); //计时器 重1970年一月一日0时0分0秒开始计时 ,他的单位是毫秒是
		作用: 用来统计我们代码的运行时间
	构造方法	
		public StringBuilder(){}  -- >  创建一个内容为空的字符串缓冲区对象
		public StringBuilder(String s){}  -->根据指定字符串创建字符串缓冲区对象
	
	StringBuilder相对于Stirng是可以变的长度  ---  它能够节省内存空间
	方法:
		length();获取字符串的长度 
		append() : 添加数据并且返回对象本身   //链式编程
		reverse() : 反转stringBuilder字符串;	
		toString() ; 将StringBuilder转成String类型
StringBuilder提高效率的原因
	StringBuilder在append方法里面不会创建新的对象而String是会创建新的对象
StringBuilder和String区别
	StringBuilder 是可变长
	Strring 是不可变长
StringBuilder和String的相互转换
	String --> StringBuilder    StringBuilder StringBuilder(String s){}
	StringBuilder --> String    toString();
